# alswl's vim configuration #

## About miv ##

miv = vim configuration

fork from [alswl/dotfiles@Github][]

## Usage ##

``` bash
git clone --recursive https://github.com/alswl/miv.git
ln -s /miv/path/.* $HOME
vim +BundleInstall +qa
```
